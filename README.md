# at-fifa-notes

本仓库用于归档和发布多个独立 HTML 页面。这些页面以静态文件形式存储，可直接通过浏览器访问，不针对任何特定域名或在线服务。

## 项目简介

at-fifa-notes 是一个轻量级的页面归档仓库。所有内容均以纯 HTML 格式保存，便于本地浏览、版本管理以及跨平台分发。仓库不依赖后端服务，仅通过 GitHub Pages 或其他静态托管方式即可展示。

## 目录说明

```
.
├── index.html          # 仓库入口页（可选）
├── pages/              # 归档页面存放目录
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── README.md           # 本文件
```

- `pages/` 目录下存放所有独立 HTML 页面，文件名按实际内容命名。
- 根目录的 `index.html` 可作为导航页，列出所有归档页面链接（非必须）。

## 页面归档说明

每个 HTML 页面均为独立、完整的文档，包含自身的样式与逻辑。页面之间互不依赖，可直接打开使用。归档内容可能包括：

- 笔记、清单、工具等各类静态页面
- 实验性或临时性的前端页面

归档遵循以下原则：

- 不包含任何具体域名、网址或商业推广信息
- 不收集用户数据或进行追踪
- 页面内容保持自包含，不依赖外部资源（或注明外部资源来源）

## 维护说明

本仓库由维护者不定期更新。如想贡献或反馈问题，请通过 GitHub Issues 提交。欢迎 Fork 后自行调整页面内容。

### 本地使用

1. 克隆仓库到本地
2. 直接用浏览器打开 `pages/` 目录下的任意 HTML 文件
3. 或通过本地 HTTP 服务器预览（如 `python -m http.server`）

### 部署建议

- 可使用 GitHub Pages 直接发布：在仓库设置中启用 Pages，选择 `main` 分支的根目录即可。
- 也可将 `pages/` 目录部署到任何静态托管平台。
